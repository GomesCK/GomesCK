# 👨🏼‍💻 Afonso Gomes

**Desenvolvedor FullStack**

Desenvolvedor Full Stack com experiência no desenvolvimento de aplicações web e APIs robustas, utilizando tecnologias como Java (Spring Boot, Spring Data, Spring Security), JavaScript/TypeScript, React, Node.js, Python e C#. Atuo com foco em soluções escaláveis, seguras e com código limpo, seguindo os princípios SOLID, Clean Code e Design Patterns.

Tenho conhecimento em bancos de dados relacionais (especialmente SQL Server), com criação e manutenção de procedures, views e scripts otimizados em T-SQL. Integro e automatizo pipelines de CI/CD utilizando Jenkins, além de empregar Docker na virtualização e orquestração de ambientes.

Atualmente, atuo como Desenvolvedor Full Stack Júnior na Datawake Digital, contribuindo com soluções voltadas ao ambiente corporativo e integração entre sistemas legados e modernos. Tenho como diferencial o aprendizado contínuo, a proatividade e o foco em entregar soluções estáveis, performáticas e com alta manutenibilidade.
